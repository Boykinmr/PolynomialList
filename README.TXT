PROJECT TITLE:			Polynomial

PURPOSE OF PROJECT:		To allow user or file input to perform addition and evaluation with polynomials.  This was to demonstrate knowledge of linked lists for class.

VERSION or DATE:		2017-11-13

HOW TO START THIS PROJECT:	Start the program from Driver program.  

AUTHORS:			Michael R. Boykin

USER INSTRUCTIONS:		First prompt gives the user the option of inputing their own parameters, opening a text file to perform the operations, or exit the program.  Polynomials are entered in the form of Ax^(n) + Ax^(n-1) + ... + Ax + constant.
